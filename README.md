# the-contactbook
can save name, e-mail, number, adress of different people.
This project is a Contact Book application built using Python that runs in the command line interface (CLI).

It allows users to store and manage contacts easily. Users can add new contacts, view saved contacts, search for contacts, and delete contacts.

This project helps beginners practice Python data structures, functions, loops, and user input handling while building a simple real-life application.
